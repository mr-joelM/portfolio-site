Gamers Parlour

the Front-end

Gamers Parlour is a place for games reviews and comments rating, and a website which interacts with the backend API I created previously. This project is a React app built using JavaScript. Using Axios and Fetch to make API calls to my Gamers Parlour Backend API and React Router to render the app's different components. The app was created with JSX styled with CSS.

Deployed on Netlify, the app can be navigated from all reviews to a more refined choice by categories.

User can read the reviews and comments from other users, and also post his own comments regarding other users reviews.

For more informations, please visit the GitHub Gamers Parlour Frontend public repo, this link will take you directly to the README.md where more details can be found regarding this project.
